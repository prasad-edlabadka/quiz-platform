# Architecture change

## Design System change
- Update the App and all the components to use Ant Design system from https://ant.design. 
- Allow theme switching between default theme and dark theme only. No other themes should be allowed.

## Add drawing tool
- If the question asks the student to draw a diagram, then add a drawing tool instead of text writing answer.
- Develop the drawing tool as per HTNML code in /sample.html. 
- All the components used in the HTML for drawing should be replicated.
- If the question asks both draw and write, answer section should allow both typing and drawing.
- AI based assessment should include diagram drawn by the user for assessing the answer.
